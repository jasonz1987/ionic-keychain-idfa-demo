# 项目说明

此项目是[cordovan-plugin-keychain-idfa](https://github.com/jasonz1987/cordova-plugin-keychain-idfa.git)插件的demo工程。



# 如何使用

```
git clone https://github.com/jasonz1987/ionic-keychain-idfa-demo.git
npm install
ionic cordova run ios
```
